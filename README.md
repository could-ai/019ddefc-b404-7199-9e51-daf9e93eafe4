# Social Network App

A responsive, basic social media application built with Flutter. Inspired by the classic Facebook feed, this app provides a clean and familiar interface for browsing posts, checking out what friends are up to, and getting ready to share your own updates.

While built as a Flutter app, it compiles beautifully to the web so you can access it directly from your phone's browser, just like a mobile website, without needing to install anything.

## Features
- **News Feed**: A scrolling feed of dummy posts with user avatars, timestamps, and images.
- **Create Post Section**: A familiar "What's on your mind?" input box to draft updates.
- **Action Buttons**: Like, Comment, and Share buttons on posts to simulate interactions.
- **Top Navigation Bar**: Facebook-style top bar with Search and Messenger icons.

## Tech Stack
- **Framework**: Flutter
- **Language**: Dart
- **Icons**: Cupertino Icons & Material Icons

## Setup and Run Instructions

To run this app locally:

1. Ensure you have Flutter installed.
2. Clone this repository and navigate to the project root.
3. Fetch the dependencies:
   ```bash
   flutter pub get
   ```
4. Run the app on web or a mobile emulator:
   ```bash
   flutter run -d chrome
   ```
